# java-demo
Spring Boot demo

"Realizar un API utilizando como framework SpringBoot, que posea los siguientes métodos:
- Un método GET que permita buscar un listado de usuarios, y que filtre opcionalmente por estado del usuario.
- Un método GET que permita buscar un usuario por su usuario Id, retornando solo un objeto usuario. El API debe poseer DTOs y Unit Tests, y se recomienda el uso de la base de datos H2 (o cualquier base de datos embebida) para el almacenamiento de los datos."
